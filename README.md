# Simple Interest Calculator

This project contains a bash shell script that calculates simple interest based on user input.

## Description
The calculator asks the user to enter:
- Principal amount
- Rate of interest
- Time period

It then calculates Simple Interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Files Included
- simple-interest.sh : Bash script for simple interest calculation
- README.md : Project documentation
- LICENSE : Apache 2.0 License

## How to Run

1. Open terminal
2. Run the script:

bash simple-interest.sh

3. Enter the required values when prompted.

## Example

Enter principal amount: 1000  
Enter rate of interest: 5  
Enter time period: 2  

Simple Interest = 100
